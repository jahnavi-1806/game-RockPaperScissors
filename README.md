#  **Rock Paper Scissors Game**

A simple and interactive **Rock Paper Scissors** game built using **HTML, CSS, and JavaScript**.
This project allows users to play against the computer with real-time score updates, hover effects, and colorful messages.

## **Features**

* Classic **Rock vs Paper vs Scissors** gameplay
* Computer-generated random choices
* Real-time score tracking
* Dynamic messages: Win, Lose, Draw
* Clean and responsive design
* Smooth hover animations


##  **How the Game Works**

* User selects **Rock**, **Paper**, or **Scissors**
* Computer randomly selects one option
* The winner is decided based on standard rules
* Scores update automatically
* A message box displays the result

##  **Technologies Used**

* **HTML5**
* **CSS3**
* **JavaScript**


##  **Project Structure**

index.html
style.css
script.js
images/


## **How to Play**

1. Open the project
2. Click on **Rock**, **Paper**, or **Scissors**
3. The computer chooses its move
4. See who wins!
5. Score updates instantly
